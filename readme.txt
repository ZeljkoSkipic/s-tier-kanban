== Description ==
A visual project management tool built right into your WordPress site. Easily track tasks, collaborate with your team, and streamline your processes.

Powerfull Kanban Boards for optimal project management, yet simplified to include only the features you will actually use, and not get confused with plethora of links, buttons and distractions.


== Installation ==
1. Upload the `s-tier-kanban` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Open the "Kanban" menu in your WordPress admin to start using it.
