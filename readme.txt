== Description ==
S-Tier Kanban is a lightweight, high-performance Kanban board plugin designed to help teams manage tasks visually within WordPress. Ideal for agencies, freelancers, and product teams who want clarity and control right from their dashboard.

== Installation ==
1. Upload the `s-tier-kanban` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Open the "Kanban" menu in your WordPress admin to start using it.
